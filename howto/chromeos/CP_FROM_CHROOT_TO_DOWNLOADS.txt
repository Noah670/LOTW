How to get stuff from Debian (bullseye) chroot into Downloads (or MyFiles).

Get into crosh, and do:
$ cd
$ ln -s /usr/local/chroots/bullseye/home/<username>/ home
$ sudo vim /usr/local/bin/remount

Type into vim:
sudo mount -o remount,exec,suid,dev,symfollow /home/chronos/user/

Now do:
$ sudo chmod +x /usr/local/bin/remount
$ remount

Now we can do:
$ cd home
